O que é JavaScript?

Em meados dos anos 90 a web se popularizou rapidamente e com o crescimento houve a necessidade de mais iteratividade na interface do cliente, o que seria possível uma linguagem de programação que fosse executada no navegador do usuário, tendo acesso ao Documento HTML da página e seus artefatos de código.

Surgiram diversas propostas na industria, mas a que se popularizou foi a linguagem script, criada na NetScape denominada de ECMA-267, conhecida popularmente como JavaScript

O código javascript é executado no navegador dentro de um sandbox, que limita as operações e acesso do código executado, fornencendo operações e variáveis limitando o acesso a rede, disco além de entrada e saída. Esse conceito é fundamental na web, em que código de terceiros podem são baixados e executados na máquina do cliente

Quando o motor o intrepetador javascript executa um script, ele disponibiliza algumas variáveis globais, que fornecem acesso ao ambiente. No node por exemplo a variável process forneece acesso a o sistema operacional e dispositivos de entrada saída. Já no navegador o javascript é executado em um sandox que tem acesso aum objeto global denominado windows, que representa a janela e tem dezenas de propriedades que fornecem acesso ao console, document, history, screen e além de funções.
